**Graph Features**

- User can view a graph with self at center
- User can see entities as nodes with distance reflecting relational health
- User can switch between three graph views (overview, gravity, reaction)
- User can tap a node to see entity detail
- User can see drift velocity per entity
- User can scrub through graph history as a timeline

**Urgency Engine**

- User can see moments sorted by urgency automatically
- Urgency is calculated from due date, gravity, drift velocity, and promise age
- User can see which moments are blocked by dependencies
- User can see which moments are blocking other moments
- User can view a ranked queue of what to do next

**Account and Settings**

- User can create an account
- User can log in
- User can log out
- User can delete their account and all data
- User can export their data
- User can set their default entity to self
- User can set notification preferences
- User can set reminder frequency

**Social and Sharing**

- User can share a moment with another Peeplist user
- User can invite someone to Peeplist from an entity profile

**Paid Tier**

- Free user can have up to 3 entities
- Paid user can have unlimited entities
- Paid user gets access to graph history timeline
- Paid user gets urgency engine
- Paid user gets crisis view
